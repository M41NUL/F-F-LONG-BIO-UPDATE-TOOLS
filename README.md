## ✨ Features

- 🎨 **36 Quick Colors** — Vivid, Dim, and Extra palettes
- 🖊️ **Rich Formatting** — Bold `[B]`, Italic `[I]`, Underline `[U]`, Strikethrough `[S]`, Center `[C]`
- 👁️ **Live Preview** — See your bio exactly as it appears in-game
- 🔐 **Multi-Token Support** — Works with Access Token, EAT Token, and JWT
- 📜 **Bio History** — Auto-saves last 10 successful bios with timestamps
- 📋 **Clipboard Tools** — Paste, Copy, and Clear in one tap
- 💫 **Smooth Animations** — iOS 26-style glass morphism dark UI
- 📱 **Mobile First** — Fully responsive, tap-friendly design

## 🛠️ Local Setup

```bash
# Clone the repo
git clone https://github.com/M41NUL/F-F-LONG-BIO-UPDATE-TOOLS.git

# Open in browser (no build step needed!)
cd ff-long-bio
open index.html
```

Or just double-click `index.html` — it works offline too (except API calls).

## 🔑 How to Get Your Token

1. Visit: [https://eat-access-token-olive.vercel.app](https://eat-access-token-olive.vercel.app)
2. Login with your Free Fire account
3. Copy your **Access Token** or **EAT Token**
4. Paste it into the tool and update your bio!
